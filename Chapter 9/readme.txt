MD3

Here is a demo that will show you how to load and render a multi-part Md3 model, complete with tags.

Files:
Md3.h - Data structures and definitions for the md3 files
Md3.cpp - Functions and other implementations to load and render md3s
Main.cpp - Main file, contains WinMain and global Init, Main Loop, and Shutdown functions.
mdl.dsp/mdl.dsw Microsoft Visual C++ 6.0 project files.

Controls:
None
