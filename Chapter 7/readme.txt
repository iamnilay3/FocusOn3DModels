3ds

This demo loads and renderes a static 3ds file

Files:
3ds.h - Data structures and definitions for the 3ds files
3ds.cpp - Functions and other implementations to load and render 3ds files
Main.cpp - Main file, contains WinMain and global Init, Main Loop, and Shutdown functions.
mdl.dsp/mdl.dsw Microsoft Visual C++ 6.0 project files.

Controls:
None
